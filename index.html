Dichos Colombianos

"Tengo un filo, que si me agacho me corto".

Esta frase es una expresión para denotar que es tanta hambre (o filo) la que se siente en el estómago, que de agacharse el cuerpo podría cortarse a sí mismo.

"Más feo que un carro por debajo”.

Es una expresión para describir una circunstancia muy desagradable

"Le falta pelo pa’ moña".

Este dicho expresa que alguien no cuenta con experiencia, pericia o recursos para lograr algo, pues le falta todavía mucho de eso para lograrlo.
